## Simple Tic-Tac-Toe game using javascript, html and css
The main functions included in the game are: 
1. initializeGame()
2. cell_clicked()
3. updateCell(cell, index)
4. changePlayer()
5. checkWinner()
6. restartGame()

These functions are utilised throughout the game and ensure the working.
